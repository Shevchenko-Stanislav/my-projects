Мои проекты

  1. 1. Agency App

    Это простой одностраничный сайт написанный на React, TypeScript и Redux Toolkit. На данной странице выводятся карточки товаров. Карточки можно фильтровать по категориям.

    Результат можно посмотреть по ссылке.

    Репозиторий: https://github.com/Shevchenko-Stanislav/agency-app.

  2. 2. Todo List

    Это планировщик дел с возможностью изменить цветовую тему. Выбранная цветовая тема и внесенные дела сохраняются в браузере.

    Результат можно посмотреть по ссылке.

    Репозиторий: https://github.com/Shevchenko-Stanislav/todo-list.

  3. 3. Адаптивная верстка 2-х страничного сайта

    Верстка 2-х страничного сайта. При нажатии на любую карточку товара вы попадаете на вторую страницу.

    Результат можно посмотреть по ссылке.

    Репозиторий: https://github.com/Shevchenko-Stanislav/Adaptive-website.

  4. 4. Fortnite Shop

    Это страница онлайн магазина. Карточки товаров генерируются динамически с помощью FortniteApi.io. Пользователь может взаимодействовать с товарами в корзине.

    Результат можно посмотреть по ссылке.

    Репозиторий: https://github.com/Shevchenko-Stanislav/fortnite-shop-react.
